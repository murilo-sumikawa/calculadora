descrição: calculadora simples usando flutter e dart

explicação de estrutura: 
- main.dart: configura a tela inicial e executa o codigo no navegador
- calculadoraPagina.dart: gerencia a lógica, organiza o layout e integra os widgets
- display.dart: mostra os numeros e resultados da calculadora
- btNumerico: componente reutilizavel para os botões com numero, recebe texto e estilo e executa função ao clicar
- btOperacao: igual ao btNumerico mas com os botões de operações
  
execução: rodar o codigo e utilizar como uma calculadora
